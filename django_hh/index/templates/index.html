ГЛАВНАЯ СТРАНИЦА

Этот сайт является порталом по поиску и найму персонала и служит ярким доказательством, что Николай Андреевич Брагин успешно постигает Вack-end с легкими вкраплениями Front-end'а
В данном проекте реализованы:
  • Отслеживание ссылок(главная страница, инфо, просмотр резюме и т.д.)

  • Работа с базой данных SQlite3

  • Минимальный фронтэнд(Разметка страниц, Bootstrap, Font-awesome)

  • Добавлена возможность регистрации пользователей, только зарегестрированные пользователи могут нанимать работников

  • Сделана обычная модель поиска, а так же модель поиска резюме по имени и по профессии (категории)

  • Добавлена возможность загрузки и отображения изображений к резюме>

  • Подключен модуль telebot: на телефон приходит сообщение, о приглашении на собеседование

  • Проект загружен на GitHub по ссылке: https://github.com/NikolayBragin/django_hh

  • Проект запущен на Pythonanywhere и доступен 24/7 по ссылке: STRsNICK.pythonanywhere.com

  • Использование API: показываю погоду в городе, где родился соискатель (Беру с сайта api.openweathermap.org)
